About Our System

Learn more about our AI-powered healthcare diagnostics platform

Project Overview

This AI-Driven Healthcare Diagnostics system leverages advanced machine learning algorithms to predict and assess various health conditions with high accuracy and speed.

Key Technologies

Machine Learning

Random Forest, SVM, Gradient Boosting, Logistic Regression

Python & Flask

Backend API and web framework

Data Science

Pandas, NumPy, Scikit-learn

Modern UI

Bootstrap 5, Responsive Design

Features

  • Heart Disease Prediction: Assess cardiovascular risk using 13 health parameters
  • Diabetes Prediction: Evaluate diabetes risk with 8 comprehensive health metrics
  • Real-time Analysis: Instant predictions with probability scores
  • Risk Level Classification: High, Medium, Low risk categorization
  • Model Training: Automated model training and evaluation
  • REST API: Programmatic access to prediction services

Model Performance

Heart Disease Model

Accuracy: ~85-90%

Trained on comprehensive cardiovascular data

Diabetes Model

Accuracy: ~80-85%

Trained on diabetes prediction dataset

Important Notice

This system is for educational and research purposes only. It should not be used as a substitute for professional medical advice, diagnosis, or treatment. Always consult with qualified healthcare professionals for medical decisions.

Getting Started

  1. Install required dependencies: pip install -r requirements.txt
  2. Run the application: python app.py
  3. Train the models using the "Train Models" button on the homepage
  4. Navigate to Heart Disease or Diabetes prediction pages
  5. Enter your health parameters and get instant predictions
